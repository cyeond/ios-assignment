# CydDesignSystem
===============

## 구조
- Colors
- Images

## Colors
- 앱 전반에서 사용할 색상 리소스를 정의합니다. 다크모드 색상은 해당 프로젝트에서 고려하지 않습니다.
    ### 구조
    - Brand  
        - 서비스를 대표하는 브랜드를 컬러를 정의
    - Common  
        - 배경색, 텍스트 색상 등의 앱 대표 색상들을 정의

## Images
- 앱 전반에서 사용할 이미지 리소스를 정의합니다.
  - ex: `CydImage.Icon.home`
